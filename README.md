Home Assistant Blueprint – Nulleinspeisung &amp; Eigenverbrauchsoptimierung für Marstek Venus E3`
 Marstek Venus E3 – Nulleinspeisung Blueprint für Home Assistant

Automatische Nulleinspeisung und Eigenverbrauchsoptimierung
für den Marstek Venus E3 Heimspeicher.

## Voraussetzungen
- Home Assistant 2024.6+
- Marstek Venus E3 in HA integriert (Modbus/RS485)
- Netzmessgerät (Tibber Pulse, Shelly EM, Eastron SDM, ...)
